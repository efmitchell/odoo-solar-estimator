How to Use

  1. Install the module from the Apps menu
  2. Open your website in the Website Builder
  3. Find "Solar Estimator" in the Features snippet section
  4. Drag it onto your page
  5. Click the snippet to customise theme, colours, and options
  6. Save and publish
